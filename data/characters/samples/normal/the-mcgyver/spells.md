### Detect Magic

| | |
|---|---|
| **School** | Divination |
| **Level** | Bard 0, Cleric 0, Druid 0, Sorcerer/Wizard 0 |
| **Casting Time** | 1 standard action |
| **Components** | V, S |
| **Range** | 60 ft. |
| **Area** | Cone-shaped emanation |
| **Duration** | Concentration, up to 1 min./level (D) |
| **Save** | None |
| **SR** | No |

You detect magical auras. The amount of information revealed depends on how long you study a particular area or subject:

- **1st round**: Presence or absence of magical auras.
- **2nd round**: Number of different magical auras and the power of the most potent aura.
- **3rd round**: The strength and location of each aura. If the items or creatures bearing the auras are in line of sight, you can make Knowledge (arcana) checks to determine the school of magic involved in each.

*The bread and butter of any wizard. McGyver uses this constantly — checking doors for traps, evaluating loot, identifying magical effects on allies and enemies. At +20 K(Arcana), he identifies most schools on the first try.*

### Prestidigitation

| | |
|---|---|
| **School** | Universal |
| **Level** | Bard 0, Sorcerer/Wizard 0 |
| **Casting Time** | 1 standard action |
| **Components** | V, S |
| **Range** | 10 ft. |
| **Target/Area/Effect** | See text |
| **Duration** | 1 hour |
| **Save** | See text |
| **SR** | No |

Prestidigitations are minor tricks that novice spellcasters use for practice. Once cast, a prestidigitation spell enables you to perform simple magical effects for 1 hour. The effects are minor and have severe limitations. You can slowly lift 1 pound of material, color/clean/soil items in a 1-foot cube, chill/warm/flavor 1 pound of nonliving material, and create small objects that look crude and artificial.

*McGyver's Swiss Army knife cantrip. Clean a wound before treating it. Warm a meal. Mark a trail with colored chalk dust. Light a fuse without flint. The practical applications are endless for someone who thinks laterally.*

### Mage Hand

| | |
|---|---|
| **School** | Transmutation |
| **Level** | Bard 0, Sorcerer/Wizard 0 |
| **Casting Time** | 1 standard action |
| **Components** | V, S |
| **Range** | Close (75 ft.) |
| **Target** | One nonmagical, unattended object weighing up to 5 lbs. |
| **Duration** | Concentration |
| **Save** | None |
| **SR** | No |

You point your finger at an object and can lift it and move it at will from a distance. As a move action, you can propel the object as far as 15 feet in any direction, though the spell ends if the distance between you and the object ever exceeds the spell's range.

*Retrieve keys from across a pit, pull a lever without stepping on the pressure plate, pass a potion to an ally 60 feet away. McGyver uses this like a third hand — because sometimes the best tool is not being there when things go wrong.*

### Light

| | |
|---|---|
| **School** | Evocation [Light] |
| **Level** | Bard 0, Cleric 0, Druid 0, Sorcerer/Wizard 0 |
| **Casting Time** | 1 standard action |
| **Components** | V, M/DF (a firefly) |
| **Range** | Touch |
| **Target** | Object touched |
| **Duration** | 100 min. (CL 10) |
| **Save** | None |
| **SR** | No |

This spell causes a touched object to glow like a torch, shedding normal light in a 20-foot radius and increasing the light level for an additional 20 feet by one step. The effect is immobile, but it can be cast on a movable object.

*100-minute duration at CL 10 means one casting covers most dungeon delves. Cast it on a coin and toss it ahead, or on your belt buckle for hands-free illumination.*

### Mage Armor

| | |
|---|---|
| **School** | Conjuration (creation) [force] |
| **Level** | Sorcerer/Wizard 1 |
| **Casting Time** | 1 standard action |
| **Components** | V, S, F (a piece of cured leather) |
| **Range** | Touch |
| **Target** | Creature touched |
| **Duration** | 10 hr. (CL 10) |
| **Save** | Will negates (harmless) |
| **SR** | No |

An invisible but tangible field of force surrounds the subject of the spell, providing a +4 armor bonus to AC. Unlike mundane armor, mage armor entails no armor check penalty, arcane spell failure chance, or speed reduction. Since mage armor is made of force, incorporeal creatures can't bypass it.

*Cast every morning — 10-hour duration covers the entire adventuring day. +4 armor bonus that stacks with Shield, works against incorporeal touch attacks, and has no downsides. This is why McGyver doesn't own actual armor.*

### Grease

| | |
|---|---|
| **School** | Conjuration (creation) |
| **Level** | Bard 1, Sorcerer/Wizard 1 |
| **Casting Time** | 1 standard action |
| **Components** | V, S, M (butter) |
| **Range** | Close (75 ft.) |
| **Target/Area** | One object or 10-ft. square |
| **Duration** | 10 min. (CL 10) |
| **Save** | Reflex partial (see text); **DC 19** (conjuration) |
| **SR** | No |

A grease spell covers a solid surface with a layer of slippery grease. Any creature in the area when the spell is cast must make a successful Reflex save or fall prone. A creature can walk within or through the area of grease at half normal speed with a DC 10 Acrobatics check. Failure means it can't move that round (and must then make a Reflex save or fall), while failure by 5 or more means it falls prone. The spell can also be used to create a greasy coating on an item, causing creatures that hold it to make a Reflex save each round or drop the item.

*One of the best 1st-level spells in the game, and it stays relevant at every level. Drop it under a melee brute to knock them prone (denying full attacks), or cast it on an enemy's weapon to disarm them without a combat maneuver. SR: No means it works on demons and dragons alike. DC 19 with Spell Focus.*

### Shield

| | |
|---|---|
| **School** | Abjuration [force] |
| **Level** | Sorcerer/Wizard 1 |
| **Casting Time** | 1 standard action |
| **Components** | V, S |
| **Range** | Personal |
| **Target** | You |
| **Duration** | 100 min. (CL 10) |
| **Save** | — |
| **SR** | — |

Shield creates an invisible shield of force that hovers in front of you. It negates magic missile attacks directed at you. The disk also provides a +4 shield bonus to AC. This bonus applies against incorporeal touch attacks, since it is a force effect. The shield has no armor check penalty or arcane spell failure chance.

*+4 shield bonus, blocks Magic Missile, 100-minute duration at CL 10. This + Mage Armor = +8 AC from two 1st-level spells. At level 10 the duration is generous enough to keep it up for most encounters. Keep scroll backups for emergencies.*

### Feather Fall

| | |
|---|---|
| **School** | Transmutation |
| **Level** | Bard 1, Sorcerer/Wizard 1 |
| **Casting Time** | 1 immediate action |
| **Components** | V |
| **Range** | Close (75 ft.) |
| **Target** | Up to 10 Medium creatures, no two more than 20 ft. apart |
| **Duration** | 1 rd./level (10 rds) |
| **Save** | Will negates (harmless) |
| **SR** | Yes (harmless) |

The affected creatures or objects fall slowly. Feather fall instantly changes the rate at which the targets fall to a mere 60 feet per round (equivalent to the end of a fall from a few feet), and the subjects take no damage upon landing while the spell is in effect. The spell affects up to 10 creatures and/or objects.

*Immediate action, verbal-only component, saves the entire party from a 200-foot drop. This is the spell you always want prepared and never want to need. Classic McGyver insurance.*

### Protection from Evil

| | |
|---|---|
| **School** | Abjuration [good] |
| **Level** | Cleric 1, Paladin 1, Sorcerer/Wizard 1 |
| **Casting Time** | 1 standard action |
| **Components** | V, S, M/DF |
| **Range** | Touch |
| **Target** | Creature touched |
| **Duration** | 10 min. (CL 10) |
| **Save** | Will negates (harmless) |
| **SR** | No (see text) |

This spell wards a creature from attacks by evil creatures, from mental control, and from summoned creatures. It creates a magical barrier around the subject at a distance of 1 foot. The barrier provides a +2 deflection bonus to AC and a +2 resistance bonus to saves, but only against attacks made or effects created by evil creatures. The subject also gains the ability to suppress mental control effects, and summoned creatures cannot physically touch the warded creature.

*The mental control suppression is the key feature. Against dominate, charm, and possession effects, this spell gives the target a new save with a +2 morale bonus. Essential against evil outsiders and undead with mind-control abilities. The +2 AC/saves vs evil is a bonus.*

### Glitterdust

| | |
|---|---|
| **School** | Conjuration (creation) |
| **Level** | Bard 2, Sorcerer/Wizard 2 |
| **Casting Time** | 1 standard action |
| **Components** | V, S, M (ground mica) |
| **Range** | Medium (200 ft.) |
| **Area** | Creatures and objects within 10-ft.-radius spread |
| **Duration** | 10 rds. (CL 10) |
| **Save** | Will negates (blinding only); **DC 20** (conjuration) |
| **SR** | No |

A cloud of golden particles covers everyone and everything in the area, causing creatures to become blinded and visibly outlining invisible things for the duration of the spell. All within the area are covered by the dust, which cannot be removed and continues to sparkle until it fades. Each round at the end of their turn, a blinded creature may attempt a new Will save to end the blindness effect.

*One of the best 2nd-level spells ever printed. Blindness is devastating (50% miss chance on attacks, denied DEX to AC, -2 to most checks), and the invisible-outline effect has no save. SR: No means it works on everything. DC 20 with Spell Focus (Conjuration). The blinded condition has no HD cap, unlike Color Spray.*

### Mirror Image

| | |
|---|---|
| **School** | Illusion (figment) |
| **Level** | Bard 2, Sorcerer/Wizard 2 |
| **Casting Time** | 1 standard action |
| **Components** | V, S |
| **Range** | Personal |
| **Target** | You |
| **Duration** | 10 min. (CL 10) |
| **Save** | — |
| **SR** | — |

This spell creates a number of illusory decoys of you (1d4+3 at CL 10, so 4–7 images). These figments stay near you and move with you, mimicking your actions. Each time a creature targets you with an attack, it must randomly determine which image (including you) it hits. An image has an AC equal to 10 + your Dexterity bonus (AC 12 for McGyver). Any attack that hits an image destroys it. Images not destroyed persist for the duration.

*The wizard's best personal defense spell. 4–7 decoy images that absorb attacks. Unlike AC, Mirror Image works against touch attacks, grapples, and even some targeted spells. Cast preemptively before combat when possible — the action economy is better than casting reactively. 10-minute duration means you can pre-buff.*

### Invisibility

| | |
|---|---|
| **School** | Illusion (glamer) |
| **Level** | Bard 2, Sorcerer/Wizard 2 |
| **Casting Time** | 1 standard action |
| **Components** | V, S, M/DF (an eyelash encased in gum arabic) |
| **Range** | Personal or touch |
| **Target** | You or a creature or object weighing no more than 100 lb./level |
| **Duration** | 10 min. (CL 10) |
| **Save** | Will negates (harmless) or Will negates (harmless, object) |
| **SR** | Yes (harmless) or Yes (harmless, object) |

The creature or object touched becomes invisible. If the subject attacks any creature, the spell ends. Items dropped or put down by an invisible creature become visible; items picked up disappear if tucked into clothing or a container. Light never becomes invisible, although a source of light can become so (the light is still visible). Invisibility does not silence a creature.

*Scout, sneak, position, escape. 10-minute duration at CL 10 is generous for reconnaissance. Cast on the rogue for infiltration, or on yourself to safely observe enemy positions. Breaks on attack, so use Greater Invisibility (4th) for combat.*

### Create Pit

| | |
|---|---|
| **School** | Conjuration (creation) |
| **Level** | Sorcerer/Wizard 2, Summoner 2 |
| **Casting Time** | 1 standard action |
| **Components** | V, S, F (miniature shovel costing 10 gp) |
| **Range** | Medium (200 ft.) |
| **Effect** | 10-ft.-by-10-ft. hole, 50 ft. deep (CL 10) |
| **Duration** | 11 rds. (1 + CL 10) |
| **Save** | Reflex negates; **DC 20** (conjuration) |
| **SR** | No |

You create a 10-foot-by-10-foot extradimensional hole with a depth of 10 feet per two caster levels (maximum 30 feet, but 50 ft. at CL 10). You must create the pit on a flat, horizontal surface of sufficient size. The pit is not a natural hazard — a creature falling in takes falling damage as normal. Creatures can climb out of the pit with a DC 25 Climb check. When the spell ends, creatures in the pit rise up with the ground.

*Drop an enemy 50 feet for 5d6 falling damage, no SR, Reflex negates. The beauty is in the denial — a melee bruiser stuck in a pit for 11 rounds is out of the fight. Combine with Stinking Cloud over the pit opening for the "microwave" combo. DC 20 with Spell Focus.*

### Web

| | |
|---|---|
| **School** | Conjuration (creation) |
| **Level** | Sorcerer/Wizard 2 |
| **Casting Time** | 1 standard action |
| **Components** | V, S, M (spider web) |
| **Range** | Medium (200 ft.) |
| **Effect** | Webs in a 20-ft.-radius spread |
| **Duration** | 100 min. (CL 10) |
| **Save** | Reflex negates (see text); **DC 20** (conjuration) |
| **SR** | No |

Web creates a many-layered mass of strong, sticky strands. These strands trap creatures caught in them. Creatures that fail their Reflex save become grappled and must make a CMB or Escape Artist check to break free (DC equal to the spell DC). The web provides cover and is difficult terrain. The web is flammable (burning for 1 round, dealing 2d4 fire damage to trapped creatures).

*Area denial for 100 minutes. Anything inside is grappled or in difficult terrain, and the web provides cover. Unlike most control spells, the long duration means you can web an area, go deal with another problem, and come back. Flammable — but that's a feature if you WANT to burn out trapped enemies. SR: No.*

### See Invisibility

| | |
|---|---|
| **School** | Divination |
| **Level** | Bard 3, Sorcerer/Wizard 2 |
| **Casting Time** | 1 standard action |
| **Components** | V, S, M (talc and powdered silver) |
| **Range** | Personal |
| **Target** | You |
| **Duration** | 100 min. (CL 10) |
| **Save** | — |
| **SR** | — |

You can see any objects or beings that are invisible within your range of vision, as well as any that are ethereal, as if they were normally visible. Such creatures are visible to you as translucent shapes. The spell does not reveal the method used to obtain invisibility and does not reveal illusions or enable you to see through opaque objects.

*100-minute duration makes this a pre-buff. Cast it at the start of a dungeon crawl and forget about invisible stalkers, sneaking rogues, and phase spiders for the rest of the session. Pairs with Glitterdust — See Invisibility for you, Glitterdust to reveal invisible enemies to the whole party.*

### Haste

| | |
|---|---|
| **School** | Transmutation |
| **Level** | Bard 3, Sorcerer/Wizard 3 |
| **Casting Time** | 1 standard action |
| **Components** | V, S, M (a shaving of licorice root) |
| **Range** | Close (75 ft.) |
| **Target** | Up to 10 creatures, no two more than 30 ft. apart |
| **Duration** | 10 rds. (CL 10) |
| **Save** | Fortitude negates (harmless) |
| **SR** | Yes (harmless) |

The transmuted creatures move and act more quickly than normal. Each affected creature gains a +1 bonus on attack rolls, a +1 dodge bonus to AC and Reflex saves, and an additional attack at its highest BAB when making a full attack action. Additionally, each creature's speed increases by 30 feet (to a maximum of twice the creature's normal speed). Multiple haste effects don't stack.

*The single most impactful buff spell in the game. +1 attack, +1 AC, +1 Reflex, +30 speed, and an extra attack for every martial in the party. A 4-person party gets 4 extra attacks per round. At CL 10, Haste affects all 10 creatures for 10 rounds. Cast this on round 1 of every important fight. Use Pearl of Power (3rd) to recover it if needed.*

### Dispel Magic

| | |
|---|---|
| **School** | Abjuration |
| **Level** | Bard 3, Cleric 3, Druid 4, Sorcerer/Wizard 3 |
| **Casting Time** | 1 standard action |
| **Components** | V, S |
| **Range** | Medium (200 ft.) |
| **Target/Area** | One creature, object, or 20-ft.-radius burst |
| **Duration** | Instantaneous |
| **Save** | None |
| **SR** | No |

You can use dispel magic to end one ongoing spell that has been cast on a creature or object, to temporarily suppress the magical abilities of a magic item, or to counter another spellcaster's spell. A dispelled spell ends as if its duration had expired. You make a dispel check (d20 + CL) against DC 11 + the target spell's CL. **Targeted dispel**: one spell on the target, choosing the highest CL spell or a specific one. **Area dispel**: 20-ft. burst, one check per creature/object. **Counterspell**: as a readied action, counter a spell being cast if you identify it with Spellcraft.

*At CL 10, dispel check is d20+10. With Focused Spells, d20+14. That's a 55% chance to dispel a CL 15 caster's buff (DC 26), improving to 75% with Focused Spells. Essential for removing enemy buffs, freeing dominated allies, and suppressing problematic magic items. Always keep one prepared.*

### Stinking Cloud

| | |
|---|---|
| **School** | Conjuration (creation) [poison] |
| **Level** | Sorcerer/Wizard 3 |
| **Casting Time** | 1 standard action |
| **Components** | V, S, M (a rotten egg or cabbage leaves) |
| **Range** | Medium (200 ft.) |
| **Effect** | Cloud spreads in 20-ft. radius, 20 ft. high |
| **Duration** | 10 rds. (CL 10) |
| **Save** | Fortitude negates; **DC 21** (conjuration); see text |
| **SR** | No |

Stinking cloud creates a bank of fog like that created by fog cloud, except that the vapors are nauseating. Living creatures in the cloud become nauseated. This condition lasts as long as the creature is in the cloud and for 1d4+1 rounds after leaving. Any creature that succeeds on its save but remains in the cloud must continue to save each round.

*Nauseated is one of the most debilitating conditions in PF1e — the creature can only take a single move action per round, no attacks, no spells, no standard actions. DC 21 Fort each round they stay in the cloud, SR: No. Creatures immune to poison are unaffected. Place it over a Create Pit for the "McGyver special" — trapped, falling, and nauseated.*

### Fly

| | |
|---|---|
| **School** | Transmutation |
| **Level** | Sorcerer/Wizard 3 |
| **Casting Time** | 1 standard action |
| **Components** | V, S, F (a wing feather) |
| **Range** | Touch |
| **Target** | Creature touched |
| **Duration** | 10 min. (CL 10) |
| **Save** | Will negates (harmless) |
| **SR** | Yes (harmless) |

The subject can fly at a speed of 60 feet (or 40 feet if it wears medium or heavy armor, or if it carries a medium or heavy load). It can ascend at half speed and descend at double speed, and its maneuverability is good. Using a fly spell requires only as much concentration as walking, so the subject can attack or cast spells normally. The subject gains a bonus on Fly skill checks equal to 1/2 your caster level (+5).

*10-minute combat flight for yourself or an ally. 60 ft. speed with good maneuverability. For all-day flight, use Overland Flight (5th) instead. Cast Fly on the fighter to let them reach a flying enemy, or on yourself for aerial battlefield positioning. McGyver's Fly check: +12 skill + 5 CL bonus = +17.*

### Fireball

| | |
|---|---|
| **School** | Evocation [fire] |
| **Level** | Sorcerer/Wizard 3 |
| **Casting Time** | 1 standard action |
| **Components** | V, S, M (a ball of bat guano and sulfur) |
| **Range** | Long (800 ft.) |
| **Area** | 20-ft.-radius spread |
| **Duration** | Instantaneous |
| **Save** | Reflex half; DC 20 |
| **SR** | Yes |

A fireball spell generates a searing explosion of flame that detonates with a low roar and deals 1d6 points of fire damage per caster level (maximum 10d6) to every creature within the area. Unattended objects also take this damage. The explosion creates almost no pressure. You point your finger and determine the range (distance and height) at which the fireball is to burst. A glowing, pea-sized bead streaks from the pointing digit and, unless it impacts upon a material body or solid barrier prior to attaining the prescribed range, blossoms into the fireball at that point.

*10d6 fire damage (avg 35) in a 20-ft. radius at 800 ft. range. Not the most efficient spell at level 10, but sometimes you need to clear a room of mooks. The long range is the real selling point — no other 3rd-level blast reaches 800 feet. With Focused Spells (+4 CL), damage is still capped at 10d6 but the CL matters for SR penetration: d20+16.*

### Dimension Door

| | |
|---|---|
| **School** | Conjuration (teleportation) |
| **Level** | Bard 4, Sorcerer/Wizard 4 |
| **Casting Time** | 1 standard action |
| **Components** | V |
| **Range** | Long (800 ft.) |
| **Target** | You and touched objects or other touched willing creatures |
| **Duration** | Instantaneous |
| **Save** | None and Will negates (object) |
| **SR** | No and Yes (object) |

You instantly transfer yourself from your current location to any other spot within range. You always arrive at exactly the spot desired — whether by simply visualizing the area or by stating direction. You can bring along objects as long as their weight doesn't exceed your maximum load. You may also bring one additional willing Medium or smaller creature per three caster levels (3 at CL 10). After using this spell, you can't take any other actions until your next turn.

*Emergency extraction — 800 ft. range, verbal-only component, brings up to 3 allies along. You can dimension door behind a wall, out of a grapple, past a blockade, or out of a collapsing building. The "no actions until next turn" clause means you can't DD and then cast — plan your exit carefully. Keep a scroll backup.*

### Black Tentacles

| | |
|---|---|
| **School** | Conjuration (creation) |
| **Level** | Sorcerer/Wizard 4 |
| **Casting Time** | 1 standard action |
| **Components** | V, S, M (octopus or squid tentacle) |
| **Range** | Medium (200 ft.) |
| **Area** | 20-ft.-radius spread |
| **Duration** | 10 rds. (CL 10) |
| **Save** | None |
| **SR** | No |

This spell causes a field of rubbery black tentacles to appear, burrowing up from the floor and reaching for any creature in the area. Every creature within the area of the spell is the target of a combat maneuver check made to grapple each round at the beginning of your turn, including the round that the spell is cast. Creatures that enter the area are also automatically attacked. The tentacles do not provoke attacks of opportunity. The tentacles' CMB is equal to your caster level + 5 (**CMB +15** at CL 10). Tentacles that successfully grapple a foe deal 1d6+4 damage each round.

*No save, no SR, CMB +15 grapple on everything in a 20-ft. radius. This is one of the best 4th-level control spells. Drop it on a cluster of enemies and they're grappled — losing DEX to AC, taking damage every round, and requiring a CMB or Escape Artist check to break free. With Focused Spells, CMB becomes +19. The only weakness is that allies can get caught too — position carefully.*

### Arcane Eye

| | |
|---|---|
| **School** | Divination (scrying) |
| **Level** | Sorcerer/Wizard 4 |
| **Casting Time** | 10 minutes |
| **Components** | V, S, M (a bit of bat fur) |
| **Range** | Unlimited |
| **Effect** | Magical sensor |
| **Duration** | 10 min. (CL 10) |
| **Save** | None |
| **SR** | No |

You create an invisible magical sensor that sends you visual information. You can create the arcane eye at any point you can see, but it can then travel outside your line of sight without limit. The eye travels at 30 feet per round (300 feet per minute) if viewing an area ahead as a human would, or 10 feet per round if examining the ceiling and walls as well as the floor ahead. The eye can see in all directions. It has darkvision to a range of 60 feet.

*10-minute casting time makes this a pre-combat tool. Send the eye through a dungeon, around corners, under doors. At 300 ft/min for 10 minutes, it covers 3,000 feet of corridor. Darkvision means it works in total darkness. McGyver never walks into a room blind — the eye goes first.*

### Teleport

| | |
|---|---|
| **School** | Conjuration (teleportation) |
| **Level** | Sorcerer/Wizard 5 |
| **Casting Time** | 1 standard action |
| **Components** | V |
| **Range** | Personal and touch |

| **Duration** | Instantaneous |
| **Save** | None and Will negates (object) |

## Borrowed Spells (via Spell Study)

These spells are NOT on the wizard list. McGyver casts them using [Spell Study](features.md#spell-study-su), which requires 1 full round per spell level and expends 2 prepared wizard spells of the appropriate level. These represent the core of the Spell Sage concept — a wizard who can do what no other wizard can.

|---|---|

### Breath of Life

| | |
|---|---|
| **School** | Conjuration (healing) |
| **Level** | Cleric 5 |
| **Casting Time** | 1 standard action |
| **Components** | V, S |
| **Range** | Touch |
| **Target** | Creature touched |
| **Duration** | Instantaneous |
| **Save** | Will negates (harmless) or Will half (see text) |
| **SR** | Yes (harmless) or Yes (see text) |

This spell cures 5d8 points of damage + 1 point per caster level (max +25). Unlike other spells that heal damage, breath of life can bring recently slain creatures back to life. If cast upon a creature that has died within 1 round, that creature is restored to life with the healed hit points. A creature brought back to life through breath of life gains a temporary negative level that lasts for 1 day.

**Spell Study cost**: 5 full rounds + the spell fires. Expend 2 prepared 5th-level wizard spells.

*Important limitation: Breath of Life must be cast within 1 round of death. Spell Study takes 5 full rounds. Spell Study + Breath of Life works ONLY if you can somehow stall or if you know death is coming (e.g., massive damage incoming next round and you start casting preemptively). In practice, use Raise Dead via Spell Study instead — the 10-day window is much more forgiving.*
