# Ancillary Feats

Ancillary Feats


        Ancillary feats encompass any feats that aren’t absolutely necessary to build a core combat build. These feats might include things that improve your ability to use skills, support your allies, branch out into combat maneuvers like feinting, etc., and can be used in a modular fashion to tack on secondary abilities to your mesmerist. Unless you’re working with a very peculiar build, none of the feats in this section should be the main attraction—they should give you a few more tricks up your sleeve.
