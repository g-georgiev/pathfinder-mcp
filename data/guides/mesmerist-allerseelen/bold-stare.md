# Bold Stare

Bold Stare


Extended Stare: The range of hypnotic stare is only 30 feet, which can be quite restrictive if enemies are fleeing from you. Honestly, I would give Extended Stare a bit of an upgrade, and allow it to have some amount of scaling with mesmerist level, say, +10 feet at 1st, 5th, 10th, 15th, and 20th levels, but then again, I’m a giver. I can’t help but give. Even if your GM won’t allow any scaling, the upgrade is badly needed, so it’ll still rank green.
Intimidating Glance: It’s really hard to scale the shaken condition’s duration well using only the base Intimidate skill, which is why so many demoralize builds eventually come to rely on something like Enforcer or Signature Skill (Intimidate). At the very least, though, Intimidating Glance solves the problem of Intimidate requiring a high action cost, which still makes this feat well worth it for dedicated debuffing builds. Hypnotic Stare (Disorientation) + shaken by Intimidating Glance + sickened by a cruel weapon + the Cursed Sanction trick = a lot of penalties.


Mental Stare: If anything, the fact that mesmerists can’t use their hypnotic stare without line of sight to a creature is an argument for creating a mesmerist with darkvision, rather than an argument for this feat. It’s nice to have, but planning for being blinded seems like a poor allocation of resources.
Obliviating Stare: I’m not saying it’s not useful for intrigue applications, only that intrigue applications are rarely optimal to focus your build on.


Gaze Reflection: Gaze effects from monsters are usually bad, it’s true, but they’re nowhere near common enough to justify spending a feat on.
