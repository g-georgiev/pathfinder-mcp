# MES 410: Archetypes

MES 410: Archetypes


As late additions to the Pathfinder class canon, many psychic classes got, well, shafted on their archetypes. Occultists have some cool archetypes, as do spiritualists, but man, kineticist archetypes are mostly garbage, and the mesmerist archetypes don’t excite me all that much. With only a few exceptions, you can think of the base mesmerist chassis as the most optimal configuration for this class, but I’ll be sure to include a few builds with archetypes down in the builds section.
