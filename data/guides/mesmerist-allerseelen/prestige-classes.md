# Prestige Classes

Prestige Classes


        Because the psychic classes were introduced so late in Pathfinder 1e’s publication cycle, there are very, very few prestige classes that truly suit mesmerists. Those that do suit the class, however, actually suit it quite well. We’ll start with an obvious recommendation that almost never goes awry: evangelist. The evangelist prestige class is associated with the Obedience feats (which, again, I wrote a massive guide about), which involve you performing some kind of task or obeisance thematic to your deity and receiving bonuses or special abilities in return. Evangelist is popular because you ultimately lose out on only one level of class features in exchange for an entire 10 levels of prestige class goodness; the big-ticket items in evangelist are always your deity’s three divine boons, so check On Bended Knee carefully to see which deities might suit your needs. Proctors and feysworn also deserve some mention as other prestige classes that come from the Obedience feats and grant solid bonuses. Esoteric knight is another option for mesmerists, although it leans considerably away from spellcasting and more toward straightforward combat applications. A larger HD, better Fortitude save, better armor and weapon proficiencies, bonus combat feats, and some magus-like psychic esoterica are all on the table here.
