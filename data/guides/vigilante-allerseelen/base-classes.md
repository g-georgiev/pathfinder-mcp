# Base Classes

Base Classes


      * Cavalier: I’m not all that familiar with the various cavalier orders, but you’ve got a good enough Charisma modifier that this variant multiclass isn’t out of the question. Note, however, that variant multiclassed cavaliers don’t receive any bonus teamwork feats, which limits the utility of tactician when you’re already trading away half of your feats.
      * Inquisitor: I suppose I’m biased, but I think the inquisitor variant multiclass is pretty solid. Judgment is strong and flexible, even 1/day or 2/day, and stern gaze synergizes naturally with many of the demoralization tactics that vigilantes are fond of. Solo tactics feels like a bit of an odd inclusion, as you run into the same problem as the cavalier variant multiclass: no bonus teamwork feats with which to fuel the class feature.
      * Oracle: It should be a mark of how powerful oracle revelations are that even the oracle variant multiclass, which is incredibly stingy with revelations, gets rated green. I’ve got a whole guide to the oracle class, so go and peruse at your leisure.
      * Alchemist: An interesting choice for experimenters, although the overlap between mutagen class features is unfortunately. Talk to your GM to see whether they’d let you get another variant multiclassing goodie at 11th level. You’ve got enormous bonuses to Craft (Alchemy) from this crossover, so prestige classes like Daggermark poisoner become more appealing. The alchemist variant multiclass is always a bit of an odd duck on Charisma-based classes, though.
      * Magus: A difficult dip for vigilantes, mostly because magus arcana, arcane pool, and spellstrike fall on deaf ears with classes that have no innate spellcasting. If you take a spellcasting archetype, though, especially one like cabalist that grants full access to the magus spell list, you’re very, very close to a magus with vigilante abilities, which is pretty cool.
      * Summoner: The summoner variant multiclass strips away most of the things that make summoners cool (standard action summoning, minutes/level duration, eidolons with buckets of evolution points) so I can’t recommend it too highly. Magical child might be the better bet.
      * Gunslinger: Bad, bad, bad. Just dip 5 levels of gunslinger if you’re so horny for firearms.
      * Witch: Oof. You’re stuck with one 1st-level hex until 15th level. Bad variant multiclass: change my mind.
