# VIG 101: Class Chassis

VIG 101: Class Chassis


Any deep understanding of a Pathfinder class has to be built on a rock-solid foundation of knowledge about what a class’ features do and don’t do. To that end, it’s become customary in all of my guides to go through the class features of each class with a fine-toothed comb, pointing out rule interactions, errata, and nuances of wording that might have large impacts on the class without being immediately obvious to novice players. Before that, though, we have to delve into what exactly the rubric is for the abilities, items, archetypes, etc. that will be reviewed in this guide.
